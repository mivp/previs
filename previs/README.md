# Previs

## Tech stack

- Node.js
- MongoDB
- Python: processing scripts
- Javascript, html, css: web client
- Electron: client app

# Install:

- Install python, node.js, mongodb
- Go to previs directory
- Run $ npm install
- Make a symbol link $ ln -s public/data mesh/

Run:
- Go to example directory
- Run $ node server.js
